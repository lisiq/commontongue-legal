# commontongue-legal

Privacy policy and legal pages for **Common Tongue** — an iOS translation-keyboard app.

Hosted via GitHub Pages at: **https://lisiq.github.io/commontongue-legal/**

This repo serves a single static `index.html` containing:
- English privacy policy
- German *Impressum* (§ 5 DDG)
- German *Datenschutzerklärung* (DSGVO)

The privacy policy URL is referenced in App Store Connect → App Privacy and in the app's Settings → Privacy screen.

## Updating the policy

Edit `index.html`, commit, push. GitHub Pages republishes within a minute or two. If anything material changes (data handling, third parties), bump the "Effective date" / "Stand:" lines.
